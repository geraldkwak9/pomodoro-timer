# ANBANG 안방

TV 시청 경험을 모티브로 한 적응형 뽀모도로 집중 시스템.
집처럼 편안한 공간에서 TV를 보듯 자연스럽게 집중과 휴식을 이어가도록 설계했습니다.
작은 고민들을 줄여 인지 부담을 낮추고, 본연의 작업에만 집중할 수 있게 합니다.

## Demo
https://geraldkwak9.github.io/pomodoro-timer/

## Features
- **CH1 · 분량 기반** 
— 단위 이름·목표 개수 설정 → 완료 시 ✓ 버튼 클릭 → 칸으로 나뉜 트랙에 반영
- **CH2 · 시간 기반** 
— 목표 시간 설정 → 타이머 실행 중 자동 누적 → 바 형태 트랙에 반영
- **자유 시청** 
— 목표 없이 집중 · 트랙 없음

- **휴식 CF** 
— 세션 종료 시 자동 휴식 화면 전환 
— 휴식 항목 커스텀
(CHANNEL 다이얼 꾹 누르기 → 휴식 항목 편집 팝업 → 이모지·텍스트 직접 수정 가능)
- **수동 튜닝** 
— 집중 흐트러질 때 다이얼 클릭 → 세션 내 3회 이상 시 집중 저하 판정 → 다음 세션 15분으로 단축
- **자막** 
— 세션 상태 변화 실시간 안내 · 1분 전 미리 안내
